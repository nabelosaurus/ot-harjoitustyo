# Harjoitustyö - Ohjelmistotekniikka

Yksinkertainen sovellus salasanojen hallintaa varten.

#### Asennus

    poetry install

#### Suoritus

    poetry run invoke start

#### Testien ajaminen

    poetry run invoke test

#### Testikattavuusraportin ajaminen

    poetry run invoke coverage-report
